# W2株式会社 入社課題

## 開発環境
- Visual Studio 2022

## 開発にあたって
- WindowsのIISを有効化してください。
参考 → https://qiita.com/portfoliokns3/items/2b1da4ffaa4669c63724
- 「コンピュータの管理」→「サービス」から"ASP.NET State Service"の設定を開き、"スタートアップの種類"を「自動」に変更してください。
- このディレクトリをC:\inetpub\wwwroot\Traning.BBSに配置してください。
- Visual Studioは管理者権限で起動してください。
- Web/w2.BBS.Front/App_Data/にある、bbs.mdf.templateとbbs_log.ldf.templateを、それぞれbbs.mdf、bbs_log.ldfにリネームしてください。
- SQL接続でエラーが発生する場合、inetmgrからアプリケーションプールの設定を開き、"Identity"を"LocalSystem"に変更してください。

## Webサイト
- w2.BBS.Front
  - 一般ユーザー向けを想定した、掲示板のWebサイトです。
- w2.BBS.Manager
  - 管理者向けを想定した、掲示板の管理画面です。

細かい要件は別途添付した資料を参考にしてください。

## ライブラリ郡について
Commonsにあるクラスライブラリは役割ごとに分けてあります。
- w2.Common
  - SQL接続や、ログ出力などの共通処理を行うライブラリです。
  - 古いコードも混じっているため、あまり参考にしないでください。
- w2.FoundationDomain
  - w2.XXXXDomainが共通して利用するロジックをまとめたライブラリです。
  - 掲示板課題では変更する必要はありません。
- w2.TemplateEngine
  - Liquidテンプレートエンジンを利用するためのライブラリです。
  - 掲示板課題では変更する必要はありません。
- w2.AccountDomain
- w2.ForumDomain
  - それぞれ、アカウント管理、掲示板のドメインロジックをまとめるライブラリです。
  - 提出段階ではこれを利用せずにWebプロジェクトに直接ロジックを書いても構いません。（入社後にリファクタリングを行います）
- w2.WebFrontDomain
  - Frontのユースケース層をまとめるライブラリです。
  - 上記同様、提出段階ではここに書かなくても構いません。

## ソフトウェアの取り扱いについて
本課題の提出物は、入社後の研修における添削・フィードバック等、研修目的の範囲内で利用させていただきます。

課題の実施に際して当社より提供するテンプレートに係るすべての権利はW2株式会社に帰属します。
また、当該テンプレートや課題内容の全部または一部を、第三者への提供、インターネット上への公開、その他再頒布することを禁止します。
